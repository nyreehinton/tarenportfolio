Image Placeholder Instructions

This file provides instructions for adding images to your Taren Mansfield portfolio website. The website has been set up with placeholder image references that you can replace with actual images.

Required Image Files

The following image files are referenced in the website. You should add these files to the images directory:

  • profile-placeholder.jpg - A profile photo of Taren (recommended size: 400x400px, will be displayed as a circle)
  • featured-placeholder.jpg - A featured/hero image (recommended size: 1200x600px)
  • work-placeholder-1.jpg - Work sample 1 (recommended size: 600x400px)
  • work-placeholder-2.jpg - Work sample 2 (recommended size: 600x400px)
  • work-placeholder-3.jpg - Work sample 3 (recommended size: 600x400px)
  • personal-placeholder-1.jpg - Personal photo 1 (recommended size: 600x400px)
  • personal-placeholder-2.jpg - Personal photo 2 (recommended size: 600x400px)
  • personal-placeholder-3.jpg - Personal photo 3 (recommended size: 600x400px)

How to Add Images

To add your own images:

  1. Prepare your images in the recommended sizes
  2. Name them according to the list above or update the references in the HTML file
  3. Place them in the images directory
  4. Update the captions in the HTML file as needed

Adding More Images

If you want to add more images to the galleries, you can copy and paste the gallery item HTML structure:

<div class="gallery-item">
    <img src="images/your-image-filename.jpg" alt="Description of image">
    <div class="gallery-caption">Your caption here</div>
</div>
